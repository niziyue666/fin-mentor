## §1 三维共振：最强买入信号

**触发条件：** Q1多头排列 + Q2财务评分≥75 + Q3情绪看多

### 什么是三维共振

技术面、财务面、情绪面三个维度同时指向同一方向，叫三维共振。这是置信度最高的信号，因为三个独立维度的信息互相印证，不是单一指标的孤立判断。

### 为什么三维共振比单一维度可靠

每个维度都有盲点：技术面只看价格，看不到公司基本面；财务面反映的是过去的数据，不代表当前市场情绪；情绪面容易被短期事件干扰。三个维度同时看多，说明当前价格既有基本面支撑，又有市场认可，还有资金流入。

### 评级输出规范

评级：**买入**

必须输出的三项内容：
1. 三个维度各自的支撑依据（不能只说"三维共振"，要具体说每个维度的数据）
2. 建议入场的价位参考（当前价或回踩支撑位）
3. 止损条件（通常是跌破MA20）

### 注意事项

三维共振是最强信号，但不是"无脑买入"的理由。仍然需要：
- 说明当前估值是否合理（PE是否处于历史高位）
- 说明当前技术位是否适合入场（不要在RSI>70时追高）

## §2 强势整理低吸：高基本面+技术回调

**触发条件：** Q1强势整理 + Q2财务评分≥75 + Q3情绪观望或看多

### 强势整理低吸的逻辑

财务顶尖的公司在上涨趋势中出现技术回调，是相对安全的买入窗口。逻辑是：好公司的股价长期由基本面决定，短期技术回调是价格暂时偏离价值，提供了以相对合理价格买入的机会。

### 低吸的前提条件（必须同时满足）

- 回调属于缩量（量比 < 1.0），说明卖盘不强，是自然整理而非主力出货
- 股价未跌破MA20（中期趋势生命线仍然完好）
- 没有出现重大基本面变化（无重大利空公告）

如果回调时放量，需要重新评估，可能不是整理而是趋势转弱。

### 分批低吸的仓位安排

分批的意义是：在不确定性尚未完全消除时，用分仓控制风险。

**标准分批方案（三批）：**

```
第一批：30%仓位
  条件：当前价在MA5与MA20之间，缩量整理
  目的：先建立基本仓位，避免错过主升行情

第二批：30%仓位
  条件：股价企稳，出现放量阳线，或RSI从低位回升
  目的：确认整理结束，加大仓位

第三批：剩余仓位
  条件：突破前期高点且放量确认
  目的：趋势确认后满仓跟进
```

### 止损条件

- 跌破MA20：中期趋势转弱，整理失败，离场观望
- 出现重大利空公告：不论价格位置，先离场再判断

### 对初学者的含义

分批低吸不是"越跌越买"，是在有明确止损条件的前提下，利用回调建立仓位。关键是第一批买入之前就要想好：跌到哪里就承认判断错了、止损离场。没有止损计划就不要执行低吸策略。

## §3 信号矛盾处理：财务好但情绪有黑天鹅

**触发条件：** Q2财务评分≥75 + Q3情绪观望 + Q3结构=黑天鹅型

### 矛盾的本质

好公司遇到突发事件，是最让初学者困惑的场景：财务面说可以买，情绪面说有风险，到底听哪个？

答案是：**短期听情绪，中长期听财务**。

黑天鹅事件的特点是不确定性极高——你不知道这个事件会对公司造成多大的实质影响，也不知道市场情绪会持续多久。在这种不确定性窗口里，财务数据是过去的，无法反映未来的变化。

### 黑天鹅事件的三个阶段

```
第一阶段：事件爆发（0-3天）
  特征：股价急跌，成交量放大，情绪极度恐慌
  建议：不要急于买入，等待靴子落地

第二阶段：信息消化（3-14天）
  特征：股价企稳或继续阴跌，成交量萎缩，市场等待后续信息
  建议：密切跟踪事件进展，观察公司官方回应

第三阶段：方向选择（14天后）
  特征：事件影响逐渐明朗，股价出现明确方向
  建议：根据事件实质影响重新评估，若基本面未受损则考虑介入
```

### 判断黑天鹅是否构成实质性威胁

以下情况说明事件影响可能持续：
- 涉及公司核心业务或核心管理层
- 监管介入且范围不明确
- 公司未发布明确的澄清或说明

以下情况说明事件影响可能有限：
- 公司及时发布详细澄清公告
- 事件属于个人问题，不涉及公司经营
- 行业监管环境未发生系统性变化

### 评级输出规范

评级：**观望**（不论财务评分多高，黑天鹅未明朗前不给买入）

必须输出的内容：
1. 明确说明为什么财务好但评级不是买入（黑天鹅逻辑）
2. 列出重新评估的触发条件（什么情况下可以升级为买入）
3. 如果用户已持仓，给出持仓建议（通常是继续持有但严守止损）

### 对初学者最重要的一句话

好公司遇到黑天鹅，不是"跌了可以抄底"，而是"先等靴子落地再说"。等待不是懦弱，是在不确定性高的时候保存子弹。

## §4 技术与基本面背离：空头排列但财务优质

**触发条件：** Q1空头排列或死叉 + Q2财务评分≥75

### 为什么好公司也会出现技术空头

财务数据反映的是过去的经营结果，技术走势反映的是当前市场的集体判断。当两者背离时，通常意味着市场正在给公司重新定价，可能的原因包括：
- 行业整体估值收缩（整个行业都在跌）
- 市场担忧公司未来成长性下降
- 短期有特定利空压制

### 好公司技术空头时的正确姿态

**错误做法：** 因为"公司好"就逆势建仓，忽视技术面的空头信号
**正确做法：** 等待技术面出现转多信号（MA5重新上穿MA20）再考虑入场

逆势买入的代价是：即使公司基本面好，股价也可能在技术空头趋势中持续下跌数月，持仓体验极差，容易在底部因为亏损焦虑而割肉。

### 等待什么信号才能入场

同时满足以下条件时，可以重新评估：
- MA5上穿MA20（技术趋势转多）
- 成交量同步放大（趋势转变有量能支撑）
- RSI从低位回升至40以上

### 评级输出规范

评级：**观望**

必须说明：
1. 当前技术空头的具体数据（MA5与MA20的差距、RSI位置）
2. 重新入场的明确技术条件
3. 不要说"可以分批低吸"——技术空头时没有低吸依据

## §5 亏损股的评级框架

**触发条件：** Q2股票类型=亏损股

### 亏损股的评级逻辑与盈利股完全不同

对盈利股问的是"值不值得买"，对亏损股问的是"能不能活下去"和"什么时候扭亏"。这两个问题决定了评级框架必须切换。

### 亏损股的评级标准

```
Q2亏损股评分 60-75分（改善明显）：
  技术面多头  → 小仓位关注，严格止损
  技术面空头  → 观望，等技术转多

Q2亏损股评分 40-59分（改善不稳定）：
  任何技术面  → 高风险观望，不建议参与

Q2亏损股评分 < 40分（生存能力存疑）：
  任何技术面  → 回避
```

### 亏损股评级必须输出的内容

1. 账上现金还能支撑多少个季度（生存能力）
2. 亏损收窄的趋势是否稳定（不是单季改善就算稳定）
3. 明确说明：这是高风险品种，适合有经验的投资者小仓位参与

### 强制免责声明

每次给亏损股评级，必须在结论末尾输出：
> "亏损股波动大、风险高，以上分析仅供参考，不构成投资建议。初学者建议观望。"

## §6 财务质量不达标：低分股的评级原则

**触发条件：** Q2财务评分 < 60

### 为什么财务评分低于60要直接回避

财务评分低于60分，通常意味着盈利能力、成长能力、资产质量中至少有两个维度存在明显问题。这类公司即使技术面出现买入信号，也缺乏基本面支撑，"技术面好看但财务差"的股票往往是游资炒作标的，波动极大、风险极高。

### 唯一的例外情况

如果公司正处于明确的困境反转期（连续多季度改善、有明确的扭亏计划），可以降低回避门槛至50分，但必须：
- 说明反转依据
- 仓位不超过总资金的5%
- 设置严格止损

### 评级输出规范

评级：**回避**

不需要提供入场建议，只需要说明：
1. 哪些财务维度不达标
2. 如果用户已持仓，给出止损建议

## §7 情绪钝化：负面消化后的底部识别

**触发条件：** Q3情绪结构=钝化型 + Q2财务评分≥60

### 什么是情绪钝化

负面消息持续发酵超过3天后，市场对这个消息的反应越来越弱——第一天大跌，第二天小跌，第三天几乎不跌了，成交量也萎缩。这叫情绪钝化，说明持续下跌的动能在衰竭。

用生活类比：就像一个坏消息刚听到时很震惊，反复听了几天之后就慢慢接受了，不再有强烈反应。

### 情绪钝化是底部信号但不是买入信号

钝化说明跌势在减弱，但不代表马上要涨。需要额外的确认信号才能入场：
- 出现放量阳线（买盘开始活跃）
- RSI从低位回升至35以上
- 技术面止跌形态（均线开始走平）

### 评级输出规范

评级：**小仓位关注**（不是买入，是开始关注）

必须输出：
1. 情绪钝化的具体表现（成交量变化、价格跌幅收窄）
2. 等待的具体确认信号
3. 如果用户想介入，第一批仓位不超过20%

## §8 三维共振回避：最强回避信号

**触发条件：** Q1空头排列 + Q2财务评分<60 + Q3情绪看空

### 三维共振回避的含义

技术面、财务面、情绪面三个维度同时指向下跌，是置信度最高的回避场景。这种情况下，没有任何理由持有或买入，唯一正确的做法是离场观望。

### 如果用户已经持仓

已持仓面对三维共振回避信号时：
- 不要等待反弹再卖，越等损失越大
- 分批卖出比一次全卖更容易执行（心理层面）
- 不要用"等回本再卖"的思维——市场不欠你一个回本的机会

### 评级输出规范

评级：**回避**

语气要明确，不要模糊表达。不要说"可以观望等待机会"，要说"当前不建议持有，建议离场"。

## §9 异常波动日的评级处理

**触发条件：** 单日涨跌幅超过±5%

### 为什么异常波动日不适合出评级

单日涨跌超过5%，通常背后有具体原因（公告、政策、事件）。这一天的技术指标、情绪数据都已经被异常波动污染，基于这些数据得出的评级结论可靠性大幅下降。

### 处理规则

**强制在评级结论前输出：**
> "⚠️ 今日异常波动（X%），当前评级结论仅供参考，可靠性低于正常交易日。建议明日数据稳定后重新评估。"

评级结论仍然输出，但加注"待确认"标记。

### 异常波动后的操作建议

不论涨跌方向：
- 不要在异常波动日做重大买卖决策
- 等待1-2个交易日，让情绪和数据稳定
- 跟踪是否有公告解释波动原因

## §10 仓位管理：分批策略与止损纪律

**触发条件：** 技术处于强势整理，或情绪结构存在不确定性

### 为什么需要分批策略

任何分析都有不确定性。分批建仓的本质是：用较小的初始仓位换取犯错的空间。第一批买错了，损失可控；第一批买对了，后续批次继续加仓放大收益。

### 仓位管理的基本框架

```
总仓位上限设定：
  单只股票不超过总资金的30%（分散风险）
  不确定性高时（黑天鹅/技术空头）不超过15%

分批节奏：
  第一批：仓位上限的1/3，条件成立时建立
  第二批：仓位上限的1/3，确认信号出现时加仓
  第三批：仓位上限的1/3，趋势明确后满仓

止损设定：
  技术止损：跌破MA20离场（中期趋势失效）
  事件止损：出现重大利空公告，不论价格位置先离场
  亏损上限：单次交易最大亏损不超过总资金的5%
```

### 止损的执行原则

止损是在买入之前就设好的，不是亏了多少之后才想。买入时就要知道：跌到哪里说明判断错了，这个价格就是止损价。

止损后不要立刻再买回来。止损意味着当前判断失效，需要重新观察，等出现新的明确信号再考虑入场。

### 对初学者的含义

很多初学者觉得设止损是认输，觉得"再等等会涨回来的"。实际上，不设止损的结果往往是小亏变大亏，最终亏损超过自己能承受的范围。

止损不是认输，是在错误方向上及时止损，把资金留下来等待更好的机会。

## 附录：Q4输出必须回答的三个问题

无论触发哪个章节，Q4的输出必须明确回答：

**第一：现在该怎么办？**（给出明确评级，不要模糊表达）

**第二：为什么这样判断？**（引用Q1/Q2/Q3的具体数据支撑结论，不能只说"综合来看"）

**第三：什么情况下要改变判断？**（给出重新评估的触发条件，让用户知道要跟踪什么）
