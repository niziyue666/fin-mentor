## §1 黑天鹅事件处理框架

**触发条件：** 新闻含监管介入/高管被查/重大诉讼等关键词，或情绪结构判断为黑天鹅型

### 什么是黑天鹅事件

黑天鹅是指突然出现、难以预测、对股价有重大影响的负面事件。在个股层面，常见的黑天鹅包括：

- 高管被监察机关留置或调查
- 公司涉及重大诉讼或监管处罚
- 核心业务突然被叫停
- 财务造假被发现

这类事件的特点是：**不确定性极高，影响持续时间不可预判**。

### 为什么黑天鹅时技术分析失效

技术分析基于"历史规律会重复"的假设，但黑天鹅事件改变了公司的基本面预期。在基本面发生重大不确定性的情况下，均线、RSI、支撑位这些技术指标的参考价值会大幅降低。

就像一辆车的导航，在正常路况下很准；但如果前方突然塌方，导航显示的路线就没用了，需要先弄清楚塌方的范围再决定怎么走。

### 黑天鹅事件的处理顺序

```
第一步：确认事件性质
  → 是否有官方公告？官方表述是什么？
  → 是个人问题还是公司经营问题？
  → 是否影响公司核心业务的正常运营？

第二步：判断影响范围
  → 一次性冲击（如个别高管问题）：对公司基本面影响有限
  → 系统性问题（如财务造假、核心业务违规）：影响深远

第三步：观察市场反应
  → 股价已大幅下跌且成交量放大：市场在消化冲击，可能已定价
  → 股价小跌但成交量异常放大：市场仍在出货，风险未释放完

第四步：等待靴子落地
  → 在官方有明确处理结果之前，技术面的任何反弹都不可信
  → 等待后续公告：是否有新的高管变动、是否影响业务资质
```

### 高管被查的历史规律

高管被查后的股价走势，历史上呈现以下规律：

- **短期（1～2周）**：股价通常承压，不确定性压制情绪
- **中期（1～3个月）**：取决于是否有后续牵连，若只是个人问题往往逐步修复
- **长期**：若公司基本面未受实质影响，股价通常回归基本面

但以上规律不能当成操作依据，每个案例情况不同。

### 对初学者的操作含义

- 黑天鹅事件期间，**不要抄底**，不要认为"跌了这么多肯定要涨回来"
- 已持仓者：先查清楚事件性质，若影响基本面应考虑减仓控风险
- 未持仓者：等待事件明朗后再做决策，错过这次机会不要紧

### 报告强制输出内容

触发此条件时，分析开头必须输出：

> ⚠️ 检测到重大突发事件（[事件描述]），当前技术面分析参考价值降低。建议优先关注后续官方公告，在事件明朗之前保持观望。

## §2 趋势型利空分析

**触发条件：** 多条利空新闻在一段时间内持续累积，情绪结构判断为趋势型

### 趋势型利空的特征

不是单一重磅利空，而是多个小利空在一段时间内接连出现，形成持续的负面情绪压力。比如：

- 行业政策持续收紧
- 公司订单数据连续几个月低于预期
- 竞争对手持续抢占市场份额的报道

趋势型利空的危险性在于它是**渐进式的**，每条单独看都不严重，但叠加在一起会形成明显的下行压力，且持续时间比单一黑天鹅更长。

### 趋势型利空与黑天鹅的区别

| | 黑天鹅 | 趋势型利空 |
|--|--------|----------|
| 来源 | 单一突发事件 | 多条持续累积 |
| 不确定性 | 极高（不知道影响多大）| 相对可预判 |
| 股价反应 | 通常急跌 | 通常阴跌 |
| 持续时间 | 等事件明朗 | 可能持续数月 |
| 恢复条件 | 官方澄清或事件平息 | 利空因素消除或出现反转信号 |

### 对初学者的操作含义

趋势型利空期间，不要轻易抄底。趋势型利空最常见的误判是"跌了这么久肯定快反弹了"——但趋势型利空的结束需要利空因素本身消除，不是靠时间等来的。

关注什么时候利空因素可能消除，而不是关注股价跌了多少。

## §3 博弈型情绪分析

**触发条件：** 利空和利好新闻数量相当，市场存在明显分歧

### 什么是博弈型情绪

博弈型情绪是指市场上同时存在较多利好和利空因素，多空双方存在明显分歧，短期方向不明确。

这种情况下股价往往表现为震荡——既不能持续上涨，也不能持续下跌，在某个区间内来回波动。

### 博弈阶段的市场特征

- **成交量放大**：分歧越大，换手率越高，双方都在积极操作
- **日内波动大**：早盘涨下午跌，或者高开低走，反复震荡
- **技术指标频繁发出矛盾信号**：均线多头但RSI超买，或均线死叉但RSI超卖

### 博弈阶段的操作策略

博弈阶段最忌讳的是**重仓押注一个方向**。在方向不明确时，轻仓或观望是最合理的选择。

等待博弈结束的信号：

- **放量突破某一关键位**：多方获胜，上涨方向确认
- **放量跌破某一支撑位**：空方获胜，下跌方向确认
- **成交量萎缩后再择向**：双方疲惫，随后的方向选择更可信

### 对初学者的操作含义

博弈阶段是老手的游戏，初学者很难在这种环境里稳定盈利。最好的策略是等方向明确后再参与，宁可少赚一段，不要在不确定时期冒险。

## §4 情绪钝化与见底信号

**触发条件：** 负面情绪已持续发酵超过3天，情绪结构判断为钝化型

### 什么是情绪钝化

情绪钝化是指市场对某条负面消息的反应越来越小——第一天大跌，第二天小跌，第三天跌幅继续收窄，说明市场已经逐渐把这个坏消息消化（定价）进去了。

就像一个坏消息说了三遍，第一遍让人很震惊，第三遍大家已经习惯了。

### 情绪钝化不等于可以马上买入

这是常见的误判。情绪钝化说明**下跌动能在减弱**，但不代表立刻会反弹。股价可以在钝化状态下横盘很久，也可以在新的利空出现时再次下跌。

### 真正的见底信号（需要多个同时出现）

- 成交量在下跌末期持续萎缩（恐慌盘释放完毕）
- 出现放量止跌（大量买盘进场承接）
- 技术面出现底背离（股价创新低但RSI没有创新低）
- 负面新闻出现但股价不再下跌（利空钝化完成）

### 对初学者的操作含义

情绪钝化是开始"观察"的信号，不是"买入"的信号。等上面至少两个见底信号出现，再考虑小仓位试探。

## §5 信息真空期判断

**触发条件：** 去重后有效新闻数量 < 3条

### 信息真空期的含义

新闻极少，说明市场对这只股票的关注度很低，或者公司近期没有值得报道的动态。这种情况下，情绪分析的可信度会显著降低。

样本太少的情绪判断，就像只凭一两个人的意见来判断一件事，结论不够可靠。

### 信息真空期的两种性质

**正常的低关注期：**
公司没有重大事件，处于平静经营期。这时股价往往跟随大盘和行业走，个股情绪意义不大。

**异常的信息真空：**
平时新闻较多，突然变得很少。这可能意味着：市场在等待某个重要公告，或者公司主动控制了信息流出。需要关注是否有未公告的重大事项。

### 报告处理规则

有效新闻 < 3条时，情绪分析部分必须输出：

> "当前新闻样本极少（有效新闻X条），情绪判断仅供参考，建议结合技术面和财务面综合判断，不宜单独依赖情绪分析。"

## §6 重复新闻泛滥的处理

**触发条件：** 去重前后新闻数量差异 > 60%（如原始20条去重后剩8条）

### 为什么会出现大量重复新闻

同一条新闻被多个媒体转载是常态，尤其是以下情况下转载率极高：

- 重大利好或利空事件
- 公司发布官方公告
- 知名分析师发表观点

如果不去重，同一条利空新闻被10个媒体转载，情绪分会被严重放大，导致情绪评分失真。

### 去重率高说明什么

去重率（原始条数减去保留条数的比例）反映了信息的原创性程度：

- 去重率 < 30%：信息来源较丰富，多个独立来源在报道
- 去重率 30%～60%：部分转载，属正常范围
- 去重率 > 60%：大量转载，真正独立的信息来源很少，情绪判断需降权

高去重率也可能是媒体炒作的信号——真正的信息很少，但大量渠道在放大传播。

### 去重保留原则

多条重复新闻保留哪一条：
1. 官方公告 > 主流财经媒体 > 自媒体
2. 原始报道（发布时间最早）> 转载稿
3. 内容最完整的版本

## §7 极度悲观情绪下的反向思考

**触发条件：** 情绪综合评分 < -60

### 极度悲观是反向信号吗

不一定，但值得关注。历史上，市场极度悲观的时候，往往也是反转出现前的阶段。原因是：

- 极度悲观意味着大多数能卖的人已经卖了，剩下的是有信念的持有者
- 情绪的极端化本身会触发反向力量（超跌反弹）
- 媒体的集中负面报道往往滞后于实际股价的下跌

但这不是"极度悲观就要买入"的逻辑，因为极度悲观也可能是有充分理由的（公司基本面真的出了大问题）。

### 极度悲观时的判断框架

```
第一步：悲观情绪是否有基本面支撑？
  → 有（公司真的出了大问题）：情绪可能还会持续，不要逆势
  → 只是情绪化反应（无重大基本面变化）：可以开始关注

第二步：技术面是否有止跌信号？
  → 有（缩量止跌、底背离）：情绪反转概率增加
  → 无：继续等待

第三步：资金面是否有抄底迹象？
  → 主力资金开始净流入：聪明钱在布局，可以小仓位跟进
  → 仍然净流出：机构仍在撤退，不急于入场
```

## §8 极度乐观情绪下的风险识别

**触发条件：** 情绪综合评分 > 60

### 极度乐观是卖出信号吗

同样不一定，但是需要提高警惕。极度乐观往往出现在股价已经涨了一大段之后，这时候风险在积累：

- 大量投资者已经入场，剩余的潜在买盘有限
- 媒体的集中正面报道往往出现在顶部附近
- 高情绪往往伴随高换手，获利盘随时可能出逃

### 极度乐观时的常见陷阱

**FOMO（害怕错过）效应：**
看到大家都在讨论、媒体都在推，感觉再不买就来不及了，冲动追高。

**利好出尽是利空：**
重大利好公告出来，股价反而下跌。因为利好在公告前市场已经预期，公告出来反而是兑现离场的时机。

### 对初学者的操作含义

极度乐观时，不是买入的好时机，更多是考虑分批止盈的时机。如果还没有持仓，等回调后再入场比追高安全得多。

## §9 政策类新闻的影响判断

**触发条件：** 新闻含行业政策/监管变化/税收调整等关键词

### 政策新闻的两种性质

**方向性政策（影响长期逻辑）：**
改变行业的基本规则，影响公司长期盈利能力。比如：加征消费税、限制某类经营行为、行业并购限制等。这类政策需要重新评估公司的长期估值。

**执行性政策（影响短期节奏）：**
在已有政策框架内的具体执行细则，影响短期业务节奏但不改变长期逻辑。比如：某项补贴政策的具体申报流程、监管检查通知等。这类政策通常不影响长期投资逻辑。

### 政策影响的判断步骤

```
第一步：判断是新政策还是旧政策的延续？
  → 新政策影响更大，旧政策可能已被市场定价

第二步：政策对公司的主业收入有直接影响吗？
  → 直接影响（如消费税涨价）：重新评估财务预测
  → 间接影响（如行业整体监管趋严）：关注竞争格局变化

第三步：政策对行业是普遍影响还是针对性影响？
  → 普遍影响：整个行业受损或受益，看相对竞争力
  → 针对性影响：具体看是否针对该公司的业务模式
```

### 常见的政策类利空

- **加税/加费**：直接压缩利润率
- **限制涨价**：影响提价能力
- **经营范围限制**：影响业务扩张
- **反垄断调查**：不确定性大，影响难以预判

## §10 业绩类新闻的解读方法

**触发条件：** 新闻含业绩预告/财报/盈利/亏损等关键词

### 业绩预告的类型

| 类型 | 含义 | 通常的市场反应 |
|------|------|-------------|
| 业绩预增 | 预计利润同比增长 | 利好，但要看是否已被预期 |
| 业绩预减 | 预计利润同比下降 | 利空，下降幅度决定影响大小 |
| 扭亏为盈 | 从亏损变为盈利 | 较强利好 |
| 首亏 | 原本盈利变为亏损 | 较强利空 |
| 续亏 | 继续亏损 | 视亏损收窄或扩大而定 |

### 业绩超预期 vs 业绩符合预期

市场的反应不只看绝对数值，更看**相对于市场预期的差异**：

- 业绩很好但符合预期：股价可能不涨甚至下跌（已被定价）
- 业绩一般但好于预期：股价可能上涨（超预期）
- 业绩很差但好于悲观预期：股价可能反弹（利空出尽）

这就是为什么有时候"业绩大增"但股价不涨，"业绩下滑"但股价反而涨。

### 对初学者的操作含义

看到业绩新闻，不要只看绝对数字，要问：这比大家预期的好还是差？这个逻辑比"利润增长=利好"更接近市场真实逻辑。

## §11 大盘恐慌时的个股分析

**触发条件：** 市场广度 < 0.3（超过70%的股票在下跌）

### 什么是市场广度

市场广度 = 上涨股票数 ÷ 全市场股票数。市场广度 < 0.3 意味着超过70%的股票在下跌，是典型的系统性下跌行情。

### 系统性下跌时个股分析的意义

大盘恐慌时，大多数个股会随大盘下跌，无论基本面多好。这时候分析个股的技术面意义有限，因为股价下跌的主要驱动力是系统性风险，不是个股因素。

就像海啸来了，游泳技术再好也很难不被波及。

### 大盘恐慌时的正确思路

```
不要问：这只股票的支撑位在哪里？
要问：大盘恐慌什么时候会结束？

判断大盘恐慌的缓解信号：
- 成交量在恐慌性抛售后开始萎缩
- 市场广度从低位开始回升
- 政策面出现托市信号
- 跌幅最大的权重股开始止跌
```

### 对初学者的操作含义

大盘系统性下跌时，不要硬扛，也不要因为个股基本面好就无视大盘风险。等大盘企稳后，好股票会涨回来，但硬扛的过程体验很差。

## §12 大盘亢奋时的风险识别

**触发条件：** 市场广度 > 0.7（超过70%的股票在上涨）

### 大盘亢奋的特征

超过70%的股票在上涨，往往伴随：
- 全市场成交量大幅放大
- 普涨行情，连基本面差的股票也在涨
- 投资者情绪高涨，大量新资金入场

### 亢奋行情中的风险

**估值泡沫风险：**
普涨行情中，很多股票涨价不是因为基本面改善，而是情绪驱动。情绪退潮后，这些没有基本面支撑的涨幅会快速回吐。

**追高风险：**
看到满屏涨停，很容易产生冲动追入的念头。但亢奋行情的顶部往往就在最多人追入的时候。

### 亢奋行情中的相对安全策略

- 持有基本面扎实、估值合理的个股，随行情享受涨幅
- 不追入基本面差、短期暴涨的投机性品种
- 分批止盈，不要在亢奋中贪心
