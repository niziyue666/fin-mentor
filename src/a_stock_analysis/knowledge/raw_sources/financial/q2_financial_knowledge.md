## §1 亏损股分析框架

**触发条件：** 净利润 < 0 且连续亏损 ≥ 2季度

### 为什么亏损股需要单独分析

盈利股和亏损股的分析逻辑根本不同。对盈利股，我们问"它现在赚多少"；对亏损股，我们问两件事：**它能活多久**，以及**它什么时候能扭亏**。

用餐厅打比方：盈利的餐厅看翻台率和净利润率，亏损的餐厅先看账上的钱够不够撑到旺季，再看每个月亏损在不在收窄。

### 亏损股的核心看点（优先级从高到低）

**第一：现金储备能烧多久**

用"账上现金 ÷ 每季度净亏损额"算出还能烧几个季度。

- 可烧 > 8季度：相对安全，有足够时间扭亏
- 可烧 5～8季度：需要关注，应密切跟踪亏损收窄进度
- 可烧 3～5季度：警惕，如无明显改善迹象需谨慎
- 可烧 < 3季度：高危，可能撑不到扭亏那天

**第二：经营现金流方向**

经营现金流为正，说明公司主业实际上在产生现金，亏损可能只是会计处理（如折旧、摊销）造成的账面亏损，实际经营比报表看起来健康。

经营现金流为负且持续恶化，说明主业本身在烧钱，比账面亏损更严重。

**第三：亏损收窄速度**

连续几个季度净亏损在缩小，说明公司在朝扭亏方向走。连续几个季度亏损在扩大，说明问题在恶化。

### 对初学者的操作含义

- 亏损股不适合初学者参与，风险明显高于盈利股
- 如果一定要关注，至少要满足：现金可烧5季度以上 + 亏损连续两季度收窄
- 财务评分满分上限为75分（亏损本身就是重大减分项）

## §2 单季亏损判断

**触发条件：** 净利润 < 0 但上季度为正（单季亏损）

### 单季亏损和持续亏损的区别

单季亏损不一定代表公司出了大问题，需要先判断是什么原因导致的。

**可能是正常的季节性亏损：**
某些行业有明显的季节性（如旅游、农业、零售），淡季亏损旺季盈利是正常现象。看年度累计净利润是否仍为正。

**可能是一次性支出：**
大规模并购、资产减值、重组费用会在某个季度集中计提，导致单季亏损，但不影响主业盈利能力。需要看扣非净利润是否仍为正。

**可能是趋势转坏的开始：**
如果同期营收也在下滑，且无一次性原因解释，单季亏损可能是趋势恶化的信号。

### 判断方法

```
第一步：看是否有一次性原因（公告/财报附注）
第二步：看扣非净利润是否为正
第三步：看营收同期是否也在下滑
第四步：看同行同期是否也亏损（行业性因素）
```

如果四步都没有合理解释，才需要认真考虑是否是趋势转坏。

## §3 减亏与盈利的区别

**触发条件：** 净利同比为正但净利润绝对值仍为负

### 这是最容易误导初学者的数据

净利润同比增长48%，听起来非常好。但如果去年亏损1亿，今年亏损0.52亿，同比变化确实是+48%，但公司**仍然在亏钱**。

这叫**减亏**，不是盈利。两者有本质区别：

| | 减亏 | 盈利增长 |
|--|------|--------|
| 净利润绝对值 | 仍为负 | 为正 |
| 公司状态 | 仍在亏钱 | 在赚钱 |
| 投资意义 | 改善信号，但风险仍高 | 可以用PE等指标评估 |
| 错误解读 | "增长48%，是好公司" | — |

### 正确的表述方式

报告中遇到这种情况，必须使用"亏损收窄X%"而不是"净利润增长X%"，避免误导读者。

减亏是向好的方向走，但距离真正的盈利还有距离，不能把它当成盈利股来分析。

## §4 高负债企业风险识别

**触发条件：** 资产负债率 > 70%

### 资产负债率是什么

资产负债率 = 总负债 ÷ 总资产，反映公司有多少资产是借来的。

70%的资产负债率意味着公司每100块资产里有70块是欠别人的。

用房子打比方：你买了100万的房子，首付30万，贷款70万，你的资产负债率就是70%。对个人来说这很常见，但对企业来说，超过70%就需要警惕了，因为企业的收入波动比个人工资大得多。

### 高负债的连锁风险

```
资产负债率 > 70%
        ↓
利息支出大 → 侵蚀利润
        ↓
银根收紧时 → 续贷困难
        ↓
流动性危机 → 被迫低价出售资产
        ↓
资产缩水 → 净资产下降 → 负债率进一步升高
```

这是一个负向循环，一旦触发很难自救。

### 例外情况

有几类行业负债率天然偏高，不能用同一标准衡量：

- **银行/保险**：负债率普遍90%以上，因为吸收存款本身就是负债，是正常经营模式
- **房地产**：70%～80%在行业内属正常，但需结合短期债务结构判断
- **建筑工程**：项目垫资模式导致负债率偏高

非以上行业，资产负债率 > 70% 触发强制警告，财务评分总分上限锁定60分。

### 对初学者的操作含义

高负债不代表公司一定会出问题，但它缩小了公司的容错空间——经济好的时候没事，一旦行业下行或信贷收紧，高负债公司受到的冲击会比低负债公司大得多。

---

## §5 负债偏高的偿债能力分析

**触发条件：** 资产负债率 50%～70%

### 负债偏高但未到危险线的判断

资产负债率在50%～70%之间，需要进一步看**债务结构**，不能只看一个总比例。

**流动比率（流动资产÷流动负债）：**
- > 2：短期偿债能力强
- 1～2：一般，需关注
- < 1：短期偿债压力大，可能面临流动性风险

**有息负债占比：**
并非所有负债都需要付利息。应付账款、预收款等是经营性负债，不需要付利息；银行贷款、债券是有息负债，会持续消耗现金。有息负债占总负债比例越高，财务压力越大。

**经营现金流覆盖利息的倍数：**
经营现金流 ÷ 利息支出，这个比值越高说明公司越有能力还利息。低于2倍需要注意。

## §6 高估值泡沫风险判断

**触发条件：** PE > 历史80%分位

### 历史高估值意味着什么

PE处于历史高位，说明当前股价相对于盈利水平是偏贵的。市场愿意给高PE，通常是因为：

- 预期公司未来增速很高（为未来增长提前买单）
- 市场整体情绪亢奋（估值虚高）
- 行业景气度处于高峰（可能即将回落）

这三种情况性质完全不同，需要结合基本面判断。

### 高估值的危险场景

**危险：高PE + 增速下滑**
市场给高PE是因为预期高增长，如果增速开始放缓，PE会双杀——既然增速下来了，市场给的PE倍数也会下来，股价跌幅会远大于利润下滑幅度。

**相对安全：高PE + 增速仍高**
如果业绩增速持续超预期，高PE可以被未来的业绩增长消化，不一定会崩。

**判断方法：**
看PEG = PE ÷ 净利润增速。PEG < 1 说明即使PE高，增速也能支撑；PEG > 2 说明估值明显高于成长性。

### 对初学者的操作含义

PE处于历史80%分位以上，不建议重仓追入。已持仓者可以考虑逢高分批减仓，等估值回归合理区间再重新评估。

## §7 低估值价值陷阱识别

**触发条件：** PE < 历史20%分位

### 低PE不等于便宜

这是很多初学者会踩的坑——看到PE很低就觉得"捡到便宜了"，但低PE有两种完全不同的情况：

**真正的低估：**
公司基本面没问题，只是市场暂时不关注或整体下跌拖累，这种低估会修复。

**价值陷阱：**
PE低是因为市场预期未来盈利会大幅下滑。比如某公司今年EPS=10元，PE=8倍看起来很便宜；但如果明年EPS可能降到3元，按当前股价算明年PE就是26倍，根本不便宜。

### 识别价值陷阱的信号

以下信号出现，低PE很可能是陷阱而不是机会：

- 营收连续多季度下滑
- 所在行业处于长期衰退期（如传统媒体、某些制造业）
- 大股东或高管在持续减持
- 经营现金流持续为负

### 对初学者的操作含义

看到低PE不要冲动买入，先问自己：这个公司的盈利能持续吗？如果答案不确定，宁可错过也不要踩进价值陷阱。

## §8 亏损股估值方法（PS替代PE）

**触发条件：** 净利润 < 0 导致PE失效

### 为什么PE在亏损时失效

PE = 股价 ÷ 每股收益。当每股收益为负数时，PE的计算结果是负数，比如股价10元、每股亏损0.5元，PE = 10 ÷ (-0.5) = -20倍。

负数的PE没有任何参考意义，不能说"-20倍PE很便宜"。

### 亏损股用PS来估值

PS（市销率）= 市值 ÷ 营业收入，用收入来代替利润做分母。因为即使公司亏损，营业收入通常是正数，PS可以正常计算。

**PS的参考标准：**
不同行业的合理PS差异很大，必须和同行业公司对比，不能跨行业比较。

- 软件/互联网：PS 3～10倍属正常范围
- 制造业：PS 0.5～2倍属正常范围
- 消费品：PS 1～3倍属正常范围

**PS的局限性：**
PS只能说明"贵不贵"，不能说明"质量好不好"——一家卖出很多货但每单都在亏钱的公司，PS可能很低，但不代表值得买。

### 报告中的处理规则

遇到PE失效场景，禁止显示PE指标，必须切换为PS，并在报告中说明原因和行业参考均值。

## §9 高PB低ROE估值不匹配

**触发条件：** PB > 3 且 ROE < 10%

### PB和ROE的匹配关系

PB（市净率）= 股价 ÷ 每股净资产，反映市场愿意为公司净资产支付多少溢价。

ROE（净资产收益率）= 净利润 ÷ 净资产，反映公司用净资产赚钱的效率。

两者应该匹配：**ROE高的公司值得高PB，ROE低的公司不应该享受高PB。**

用银行账户打比方：一个账户每年能产生10%的利息，另一个只能产生2%的利息。你愿意为前者多付钱，但不应该为后者付同样多的钱。

### PB > 3 但 ROE < 10% 的含义

市场给了这家公司3倍以上的净资产溢价，但公司用净资产赚钱的效率很低（不到10%）。这说明：

- 要么市场对公司未来的高增长有预期（当前ROE低，但预期未来会高）
- 要么市场存在非理性定价，股价虚高

可以用 ROE ÷ PB 来量化匹配程度：
- ROE/PB ≥ 3：溢价合理
- ROE/PB 1～3：需要有增长预期支撑
- ROE/PB < 1：溢价明显高于盈利能力，需警惕

---

## §10 高PB被高ROE支撑的判断

**触发条件：** ROE/PB ≥ 3

### 高PB合理的情形

高PB不一定意味着贵。如果一家公司ROE持续在20%以上，市场给3倍PB是合理的，因为这家公司的净资产能以很高的效率持续增值。

以茅台为例：PB=7倍看起来很贵，但ROE=26%，ROE÷PB=3.7，说明每一块钱的净资产溢价，背后有足够的盈利能力支撑。

### 判断高PB是否合理的完整逻辑

```
第一步：看ROE是否持续（连续3年以上ROE > 15%）
第二步：算ROE/PB比值（≥3为合理，1～3需要增长支撑，<1需警惕）
第三步：看护城河是否稳固（品牌/专利/规模效应是否能维持高ROE）
```

三步都通过，高PB才算站得住脚。

## §11 营收负增长衰退期分析

**触发条件：** 营收增速 < 0

### 营收负增长意味着什么

营收是公司最基础的生命体征。净利润可以通过压缩成本临时提升，但营收负增长说明公司的业务规模在收缩，是更根本的问题。

营收负增长的可能原因：
- **行业性衰退**：整个行业在萎缩，公司随行业下滑
- **竞争失利**：行业没问题，但公司丢失了市场份额
- **主动收缩**：剥离低利润业务，主动做小换利润（这种情况需结合利润率是否提升判断）
- **周期性回落**：上一年基数高，今年自然回落（需看两年复合增速）

### 对初学者的操作含义

营收负增长时，不要被"利润增长"迷惑。如果利润增长是靠裁员、卖资产、压缩研发实现的，而不是靠主业扩张，这种利润增长是不可持续的。

优先看营收，营收在长期持续下滑的公司，很难长期持有。

## §12 成长瓶颈期企业分析

**触发条件：** 营收增速连续3季度下滑

### 成长瓶颈的特征

营收增速连续下滑，不代表公司出问题，但代表公司进入了一个新的成长阶段。很多优秀公司都会经历从高增速到低增速的切换期。

关键是要判断：这是**暂时性放缓**还是**长期性瓶颈**。

**暂时性放缓的信号：**
- 行业整体都在放缓（宏观或政策原因）
- 公司在主动调整业务结构（短期牺牲增速换质量）
- 新产品/新市场处于蓄力期，增速会回来

**长期性瓶颈的信号：**
- 主要市场已接近饱和，没有新增量
- 竞争对手在快速抢占份额
- 公司研发投入持续减少

### 估值重估

增速从高增长切换到低增长，市场给的PE倍数也会相应下调。比如原来给30倍PE是因为增速30%，现在增速降到5%，30倍PE就显得太贵了，估值可能面临压缩。

## §13 应收账款风险识别

**触发条件：** 应收账款占营收比 > 50%

### 应收账款是什么

应收账款是公司已经提供了商品或服务，但还没有收到的钱。账面上算作收入，但钱还没到账。

如果应收账款占营收比超过50%，说明公司超过一半的"收入"还是纸面上的数字，没有真正变成现金。

### 高应收账款的风险

**坏账风险：**
客户如果经营出现问题，这笔钱可能就收不回来了，直接变成坏账冲减利润。

**回款周期风险：**
政府/国企类客户回款周期可能长达一年以上，公司账面盈利但实际现金紧张，可能需要借钱维持日常运营。

**利润注水风险：**
极端情况下，公司可能通过虚构销售来虚增收入，应收账款会异常偏高。

### 判断是否正常

不同行业应收账款水平差异很大：
- 软件/政务类企业：客户多为政府，回款慢，50%以上属常见
- 消费品企业：通常预收或即时结算，应收账款应很低，超过20%就需要关注
- 制造业：视行业惯例，通常30%～50%属正常范围

要结合行业特性判断，不能一刀切。

## §14 利润质量验证

**触发条件：** 经营现金流/净利润 < 80%

### 为什么要用现金流验证利润

净利润是会计数字，经营现金流才是真实流入公司的现金。两者之间存在差异，但如果差异过大，需要警惕。

**经营现金流/净利润的含义：**
- > 120%：利润质量很高，现金比利润还多，说明预收款多或应收款少
- 100%～120%：正常，利润基本都转化为现金
- 80%～100%：略低，需要关注应收款是否在增加
- < 80%：利润质量偏低，大量利润停留在应收款或存货里，没有变成现金

### 低现金流比率的含义

大量利润没有转化为现金，可能的原因：

- 应收账款快速增加（赊账卖货）
- 存货积压（生产了但没卖出去）
- 预付款增加（提前付了钱但还没收到货）

这三种情况都会导致利润看起来不错，但公司口袋里没钱。

## §15 单季利润异常波动

**触发条件：** 单季净利润环比变化 > 30%

### 单季异常波动的常见原因

**正常原因（不需要担心）：**
- 季节性因素：白酒Q4旺季、零售业Q4购物季
- 上年同期基数异常（去年同期有一次性收益或损失）
- 收入确认节奏：大合同在某季度集中确认

**需要深入了解的原因：**
- 大额资产减值（一次性冲减利润）
- 政府补贴集中到账（一次性增加利润）
- 重大合同签订或终止

**需要警惕的原因：**
- 无合理解释的利润大幅下滑
- 应收账款同步大幅增加
- 经营现金流方向与利润方向相反

### 判断方法

遇到单季异常波动，第一步是查公告，看公司是否有解释。有合理解释的一次性波动不需要过度担忧，无法解释的波动才需要警惕。
